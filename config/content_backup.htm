WSGI

WSGI，全稱 Web Server Gateway Interface，或者 Python Web Server Gateway Interface ，是為 Python 語言定義的 Web 伺服器和 Web 應用程式或框架之間的一種簡單而通用的介面。這東西是一個Gateway，也就是閘道器。閘道器的作用就是在協議之間進行轉換。
